# Tris-Tic-Tac-Toe-

Project developed entirely in Java, implementing the classic Tic-Tac-Toe game with a graphical interface. The program allows two players to compete in a match, displaying an interactive 3x3 grid that updates in real time after each move.


Progetto sviluppato interamente in Java, che implementa il classico gioco del Tris (tic-tac-toe) con interfaccia grafica. Il programma permette a due giocatori di sfidarsi in una partita, mostrando una griglia 3x3 interattiva e aggiornata in tempo reale dopo ogni mossa.
